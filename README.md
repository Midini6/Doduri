# 도두리동물병원 웹사이트

우리 동네 평생 건강 지킴이, **도두리동물병원** 공식 웹사이트입니다.

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3 (Vanilla) |
| 인터랙션 | JavaScript (Vanilla) |
| 폰트 | Google Fonts — Noto Sans KR |
| 아이콘 | Font Awesome 6 |
| 배포 | Netlify |

---

## 폴더 구조

```
Doduri/
├── index.html                 # 메인 페이지
├── about-greeting.html        # 병원소개 > 인사말
├── about-doctor.html          # 병원소개 > 의료진소개
├── about-facility.html        # 병원소개 > 시설소개
├── service-subject.html       # 진료안내 > 진료과목
├── service-location.html      # 진료안내 > 진료시간 / 오시는길
├── contact.html               # 상담/예약
├── _header.html               # 공통 헤더 (JS로 삽입)
├── _footer.html               # 공통 푸터 (JS로 삽입)
├── css/
│   ├── style.css              # 전체 공통 스타일
│   └── sub.css                # 서브 페이지 전용 스타일
├── js/
│   ├── include.js             # 헤더/푸터 동적 삽입
│   ├── script.js              # 공통 인터랙션
│   └── sub.js                 # 서브 페이지 슬라이더 등
├── images/
│   └── facility/              # 시설 사진 (facility1~4.jpg)
└── logo/                      # 로고 이미지
```

## 주요 기능

### 공통
- 반응형 웹 디자인 — 모바일 / 태블릿 / PC 전 환경 대응
- 공통 헤더·푸터 분리 (`_header.html`, `_footer.html`) — `fetch()`로 동적 삽입
- 스크롤 진행바 — 페이지 상단 시각화
- 플로팅 버튼 — 전화상담 / 카톡상담 / 블로그 빠른 접근
- 맨 위로 버튼

### 메인 페이지 (`index.html`)
- 히어로 섹션 — 배경 이미지 + 병원 슬로건

### 병원소개
- **인사말** — 원장 인사말 및 진료 철학
- **의료진소개** — 원장 학력·경력·학술활동
- **시설소개** — 시설 이미지 슬라이더
  - 썸네일 클릭 / 화살표 / 마우스 드래그 / 터치 스와이프
  - 이미지 클릭 시 라이트박스 확대
  - 4초 자동재생 (호버 시 일시정지)

### 진료안내
- **진료과목** — 건강검진·예방의학 / 내과질환 클리닉 / 순환기·시니어 클리닉
- **진료시간 / 오시는길** — 지도(API 연동 예정) + 교통편·주차·진료시간·연락처

### 상담/예약 (`contact.html`)
- 준비 중

## 입력 필요 항목 (미완성)

| 항목 | 현재 상태 |
|------|-----------|
| 전화번호 | `000-000-0000` (임시) |
| 카카오채널 | `contact.html` 링크 예정 |
| 네이버 지도 링크 | 미연동 |
| 카카오 지도 링크 | 미연동 |
| 지도 API | 플레이스홀더 표시 중 |
| 의료진 사진 | 준비중 표시 |
| 점심 마감 시간 | 미입력 |
| 인스타그램 URL | 미연결 |
