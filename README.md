# 🐾 도두리동물병원 웹사이트

임상 경험을 바탕으로 지역에서도 믿고 맡길 수 있는 주치의 같은 동물병원, **도두리동물병원**의 공식 웹사이트입니다.

---

## 🛠 기술 스택

| 분류 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3 (Vanilla) |
| 인터랙션 | JavaScript (Vanilla) |
| 폰트 | Google Fonts (Noto Sans KR, Playfair Display) |
| 아이콘 | Font Awesome 6 |
| 지도 | Kakao Map |

---

## 📁 폴더 구조

```
Animal_Clinic/
├── index.html       # 메인 페이지
├── style.css        # 전체 스타일
├── script.js        # 인터랙션 스크립트
└── logo/            # 로고 이미지 파일
```

---

## ✨ 주요 기능

- 📱 **반응형 웹 디자인** — 모바일 / 태블릿 / PC 모든 환경 대응
- 🎠 **히어로 슬라이더** — 3개 슬라이드 자동재생 및 수동 전환
- 🗂 **진료과목 탭 메뉴** — 건강검진 / 내과질환 / 순환기·시니어 클리닉
- 🎬 **스크롤 애니메이션** — 섹션 진입 시 요소 페이드인 효과
- 📞 **플로팅 사이드 버튼** — 전화상담 / 카톡상담 / 블로그 / 인스타 빠른 접근
- 🗺 **카카오맵 연동** — 병원 위치 지도 표시
- 📊 **스크롤 진행바** — 페이지 상단 스크롤 위치 시각화

---

## 🚀 로컬 실행 방법

Node.js가 설치되어 있는 경우 아래 명령어로 실행할 수 있습니다.

```bash
npx serve .
```

실행 후 브라우저에서 접속:

```
http://localhost:3000
```
